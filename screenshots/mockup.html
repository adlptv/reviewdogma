Code Review Policy Engine

Define review rules as YAML. Catch anti-patterns that escape ESLint and Sonar — "No any type in new files", "Every API must have rate limiting". Integrates with GitHub Apps.

Get Started → View on GitHub
📜

Policy DSL

Write rules in YAML with custom patterns and conditions

🔍

AST Detection

Parse JS/TS/Python with tree-sitter, match against policies

🤖

GitHub Integration

Auto-post review comments on PRs with severity badges

📊

Team Dashboard

Anti-pattern trends, review coverage, common issues leaderboard

🏪

Policy Marketplace

Share and import community policies between teams

📝

Monaco Editor

YAML validation, auto-complete, real-time syntax highlighting

Quick Start

$ git clone https://github.com/adlptv/reviewdogma.git
$ cd reviewdogma
$ pnpm install
$ pnpm dev

→ Open http://localhost:3000

$ docker-compose up # or Docker
